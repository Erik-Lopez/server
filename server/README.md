# SERVER

## INICIAR EL SERVER

Para encender el server se debe ejecutar el archivo jar utilizando java:

java -jar server.jar 		//No inicializa ningún plugin
java -jar paper-1.16.5-547.jar 	//Inicializa los plugins

## SCRIPT GET-UPDATE

Cuando uno vaya a inicializar el servidor, querrá tener el mundo actualizado. Para ello debe ejecutar el script get-update.sh.

## SCRIPT SEND-UPDATE

Cuando uno vaya a apagar el servidor, querrá enviar un mundo actualizado. Para ello debe ejecutar el script send-update.sh.
